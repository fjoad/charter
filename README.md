# Charter

**AI-assisted project discipline plugin for Claude Code and Codex CLI.**

Charter enforces session-to-session continuity through living docs and lightweight rituals. You're the manager. The AI is your dev team. Charter is your PMO.

> Full README landing page coming in Step 8 of implementation. See [docs/MANIFESTO.md](docs/MANIFESTO.md) for the pitch and [docs/VISION.md](docs/VISION.md) for the thesis.

---

## Quick Start

```
/plugin install superpowers
/plugin install charter
/charter-init
```

---

## What It Does

- Scaffolds living docs (VISION, STATUS, ARCHITECTURE) into any project
- Orients Claude at session start by injecting current project state
- Classifies every request by size and applies matching ritual depth
- Enforces finish rituals — docs updated, tests run, status recorded
